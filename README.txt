How to play:

Use arrow keys to move snake (green) and grab food (magenta).
Game ends when snake moves out of playing zone, bites itself, or bites enemy snake (yellow).

Feature List:

v1.0
- adjustable playing zone size
- speed increases as snake grows

v1.1

- turbo mode

v1.2
